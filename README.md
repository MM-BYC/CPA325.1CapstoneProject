## Backend
```
1. cd backend
   * npm init -y
   * npm i express mongoose dotenv nodemon cookie-parser
    
2. create server.js
3. Edit package.json 
   * "start": "node server.js",
   * "dev"" "nodemon server.js"
4. create .env
    PORT=
    DB_URL=
    SECRET=
5. Test server is running.
   * npm start
   ![npm start to run the server](npmstart.png)
```
